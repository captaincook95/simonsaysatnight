Simon Says @ Night


Diese Webseite wird zum Lösen des Geocaches "Simon Says @ Night" (GCXXXXX) benötigt.
Die Bahnanlange muss zu keinem Zeitpunkt betreten werden! Haltet euch bitte von den Gleisen fern.
Wenn ihr zu einem Reflektor lauft, achtet ein wenig auf den Boden vor euch, teilweise gibt es kleine Löcher oder Unebenheiten

Die folgende Webseite enthält vier Kreise (Blau, Rot, Gelb und Weiß). In der Mitte wird ein Timer angezeigt, der von einer Minute herunterzählt.
Folgt der Reflektorkette und gebt jeweils die gesamte Reflektorkette bis zu eurem aktuellen Punkt ein, bevor die Zeit abgelaufen ist. An manchen Punkten befinden sich mehrere gleichfarbige Reflektoren. Diese dienen nur der besseren Sichtbarkeit und können als ein einziger Reflektor behandelt werden.
Beispiel: Nehmen wir an, der erste Reflektor ist blau. Drückt also einfach auf den blauen Kreis. Die Anwendung quittiert das mit einem "OK". Sucht nun den nächsten Reflektor. Ist dieser z.B. rot, so drückt ihr jetzt Blau-Rot.
Mit jedem Reflektor wird die einzugebende Kette um eins länger. Die gesamte Kette hat eine Länge von 7 Reflektoren. Habt ihr diese korrekt eingegeben, erhaltet ihr weitere Koordinaten.

Diese Webseite ist mit einem Code verschlüsselt. Von den Start-Koordinaten aus seht ihr einen weißen Markierungsstein mit einer Zahl (neben den Schienen unter der Unterführung). Gebt diese in das folgende Feld ein.
(Falls der Code falsch ist, startet die Anwendung trotzdem, aber eure Aufgabe wird nicht lösbar sein)
